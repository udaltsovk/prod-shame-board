# 📛 Как номинировать на доску позора

1. Открой [**New Issue**](../../issues/new?template=shame.yml)
2. Заполни форму: телеграм, цитата, ссылка на пруф
3. Отправь
4. Мейнтейнер ставит лейбл `approved` → бот автоматически создаёт PR
5. PR мёржится → запись появляется на сайте

## Правила

- ✅ Только реальные цитаты из чата
- ✅ Указывай корректный Telegram-тег автора в формате `@username`
